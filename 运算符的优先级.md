###运算符的优先级

    优先级从高到底
    ()  优先级最高
    一元运算符  ++   --   !
    算数运算符  先*  /  %   后 +   -
    关系运算符  >   >=   <   <=
    相等运算符   ==   !=    ===    !==
    逻辑运算符 先&&   后||  

###逻辑运算符

#####1.非
   
    如果操作数是对象，返回false
    如果操作数是空字符串，返回true
    如果操作数是非空字符串，返回false
    如果操作数是0，返回true
    如果操作数是任意非0值，返回false
    如果操作数是null,undefined,NaN，返回true
    上面的规则可以简化成
    转换为false的值：false、""、0和NaN、null、    undefined
